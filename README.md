# 🧩 像素拼图游戏

一个有趣的像素风拼图游戏，可以上传自己的图片并将其转换为像素风格，然后进行拼图挑战。

## ✨ 功能特点

- **📷 图片上传** - 支持上传本地图片，可裁剪为任意比例
- **🎨 像素化处理** - 实时调整像素大小，预览像素化效果
- **🧩 拼图游戏** - 完整的拼图逻辑，支持点击/拖拽移动拼图块
- **📊 难度选择** - 提供 6 种预设难度，支持自定义网格大小
- **⏱️ 计时系统** - 记录游戏用时和移动次数
- **🏆 排行榜** - 本地存储最佳成绩，按难度分组展示
- **💾 图片下载** - 一键下载像素化后的图片

## 🎮 难度级别

| 难度 | 网格 | 拼图块数量 |
|------|------|-----------|
| 入门 | 2×2 | 4 块 |
| 简单 | 3×3 | 9 块 |
| 普通 | 4×4 | 16 块 |
| 中等 | 5×5 | 25 块 |
| 困难 | 6×6 | 36 块 |
| 专家 | 8×8 | 64 块 |

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 📖 使用说明

1. **上传图片** - 点击上传按钮选择本地图片
2. **裁剪图片** - 拖动裁剪框调整图片区域和比例
3. **调整像素** - 拖动滑块调整像素化程度
4. **选择难度** - 选择预设难度或自定义网格大小
5. **开始游戏** - 点击开始按钮进入拼图模式
6. **完成挑战** - 将所有拼图块移回正确位置

## 🛠️ 技术栈

- **React 19** - 用户界面
- **Vite** - 构建工具
- **Canvas API** - 图片像素化处理
- **LocalStorage** - 排行榜数据存储
- **CSS3** - 响应式样式

## 📝 游戏规则

- 拼图会被分割成指定数量的块，其中一块显示为空白
- 点击任意相邻于空白的拼图块即可移动
- 将所有拼图块移回正确位置即可完成游戏
- 完成拼图后会显示用时和移动次数
- 成绩会自动保存到本地排行榜
